点字フォント（Unicode６点）
バージョン2.0
2008/12/26


＜機能説明＞
世界標準の文字コードUnicodeには「点字」を表すために割り当てられている領域（U+2800〜U+28FF）があります。
このフォントは、そのうち６点点字の部分（U+2800〜U+283F）のみを表示することができます。
従来から日本で使用されている点字フォントには多くの問題点がありましたが、新しい技術方式を駆使することにより根本的に改善しました。
TrueTypeという一般的なフォント形式のため、Windows以外のOS（Linux、Macintoshなど）でも使用することができます。


＜インストール／アンインストール方法＞
OSにより多少異なりますが、基本的にフォントフォルダに「Braille6.ttf」をドラッグ＆ドロップで入れるだけです。
（ただし、ユーザの権限が低いと、システムフォルダへのアクセス権がないため出来ない場合があります）
削除したい場合は、そのフォルダから「Braille6.ttf」を消してください。

【例】Windows 98、Me、2000、XP、Vista の場合
→ Braille6.ttf を コントロールパネルの「フォント」 の中に入れる。

【例】Vine Linux 4 の場合
→ Braille6.ttf を /usr/X11R6/lib/X11/fonts/TrueType の中に入れる。

【例】Mac OS X の場合
→ Braille6.ttf を Macintosh HDのシステムフォルダの「フォント」 の中に入れる。
